# ExpenseTracker

Мобильное приложение для Android, которое помогает контролировать 
личный бюджет - записывать траты и доходы, следить за категориями 
расходов и отслеживать актуальные курсы валют.

---

## Description

ExpenseTracker построен на Kotlin и Jetpack Compose с использованием 
чистой архитектуры MVVM. Приложение хранит все транзакции локально 
в базе данных Room, автоматически обновляет курсы валют через API 
Национального банка Республики Беларусь (НБРБ) в фоновом режиме 
и отправляет уведомления, когда расходы приближаются к установленному 
бюджету. Интерфейс поддерживает светлую и тёмную тему.

---

## Installation

1. Клонируйте репозиторий:
```
git clone https://github.com/DashaSidorskaya/expense-tracker.git
```
2. Откройте папку проекта в Android Studio (версия Hedgehog 2023.1.1+).
3. Дождитесь синхронизации Gradle — все зависимости загрузятся автоматически.
4. Подключите устройство или запустите эмулятор (API 26+).
5. Нажмите **Run → Run app**.

Альтернативно: готовый APK можно скачать во вкладке **Actions** → 
последний успешный запуск → раздел **Artifacts → debug-apk**.

## Usage

**Главный экран**
Показывает баланс за выбранный период (день, неделя, месяц, год),
суммарные доходы и расходы. Список транзакций обновляется в реальном 
времени при смене периода.

**Добавление записи**
Кнопка **+** открывает форму: выбор типа (расход или доход), сумма, 
категория, дата и необязательная заметка. Редактировать запись можно 
нажав на неё, удалить - смахнув влево.

**Статистика**
Вкладка Stats отображает распределение расходов по категориям 
и столбчатый график за последние 14 дней.

**Настройки**
Выбор валюты отображения, установка месячного бюджета, переключатель 
тёмной темы, таблица курсов НБРБ с кнопкой обновления.
